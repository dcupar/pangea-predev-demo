Project Description

We are looking for an experienced development team to build CrossCast, a cross-platform social audio and video chat application MVP. The platform should enable iOS, Android, and Web users to discover and join live rooms for casual interaction.

Key requirements include:

- Low-latency real-time voice and video communication (sub-200ms)
- Scalable room discovery and presence system
- Role-based moderation (User/Moderator/Admin)
- Simple social features (follow, profiles)
- Analytics for moderation

The expected outcome is a functional MVP supporting rooms with moderator controls, reliable authentication and account management, real-time media via WebRTC + SFU, push notifications, and deployment-ready infrastructure on AWS with CI/CD.
